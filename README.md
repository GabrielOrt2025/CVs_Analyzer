Sistema inteligente para el análisis automatizado de currículums, diseñado para evaluar candidatos de manera objetiva mediante el uso de inteligencia artificial.

La aplicación permite cargar CVs en formato PDF, extraer su contenido y analizar la experiencia, habilidades y formación del candidato en función de una descripción de puesto específica. A partir de este análisis, se genera una evaluación estructurada que facilita la toma de decisiones en procesos de reclutamiento.

---

## Tecnologías utilizadas

- Python  
- Streamlit  
- LangChain  
- OpenAI API  
- Pydantic  
- PyPDF2

---

## Características principales

- Análisis automático de currículums en formato PDF  
- Evaluación del nivel de ajuste del candidato al puesto  
- Identificación de habilidades clave y experiencia relevante  
- Generación de recomendaciones estructuradas  
- Interfaz interactiva y fácil de usar
